# Breast Cancer Mortality Analysis – Southeast Region (2005–2020)

This is a project developed for the Probability and Statistics course in the first semester of 2025. The repository contains an exploratory statistical analysis of the variation in breast cancer mortality rates among women, considering data from the Southeast region of Brazil between 2005 and 2020.

## 📊 Objectives

- Evaluate variations by state and by year
- Relate SDI (Sociodemographic Index) to mortality
- Analyze the influence of mammography coverage
- Examine the association with the number of hospitals

## 📁 Structure

- `projeto.R`: main code in R
- `base.xlsx`: dataset
- `README.md`: this file
